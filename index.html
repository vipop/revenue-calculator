    Title:          Total Revenue Calculator
    Version:        1.0
    Purpose:        Parse JSON to retrieve information and display it
                    as desired.
    Description:    Retrieves all orders from a JSON file and lists them
                    with their corresponding total prices.
                    Also calculates the total revenue by adding the total order
                    prices(assuming they include shipping), then displays the
                    total revenue including and excluding tax(13% in Canada).
                    
    Live Demo:      https://shop-test-web.000webhostapp.com/
    Note:           This HTML file must be placed on the same host as the JSON
                    file due to access permissions.
                    For the demo, the JSON file is located at
                    https://shop-test-web.000webhostapp.com/orders.json
